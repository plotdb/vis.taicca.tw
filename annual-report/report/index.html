簡介

此文件提供文化內容策進院展示國內文化內容創意產業相關數據之視覺化人機介面之設計 ( 文創產業視覺化，簡稱視覺化 )。此視覺化以協助查閱、理解與發掘洞見為主要目標，並以單一入口為主，透過互動、轉存與分享功能，搭配轉場動畫引導用戶閱讀並使用儀表板觀察資料。

此視覺化包含之資料範圍以 文化部文化統計資料集 其中五組重點資料為主，包括：

  • 文化創意產業家數與營業額 - 依產業別分
  • 文化創意產業家數與營業額 - 按經營年數分
  • 文化創意產業家數與營業額 - 按資本結構分
  • 文化創意產業家數與營業額 - 按縣市分
  • 文化相關產業就業人數 ( 依資料性質切分為「產業別人數」與「總體人數」兩組資料 )

介面流程

考慮本計劃處理之資料均為時序資料，因此我們以「逐年趨勢」做為視覺化呈現的首要重點，並先以一般民眾最熟悉的「折線圖」做為主圖將資料以概觀形式呈現。由時間切入，我們以下面這個順序做資訊架構。

逐年趨勢

在進入視覺化介面的同時，我們僅顯示其中一組資料 ( 如家數、營業額等 )，用戶可以自由挑選想瀏覽的數據。由於沒有解析度更高的資料，用戶必須同時選取「分組」與「數值類型」，才能呈現特定的資料集。

介面預設的資料分組為「產業別」，數值類型則為「營業額」，即介面提供給訪客的第一印象為「文創類各產業逐年營業額趨勢」。以下圖為例，可見到圖表中有「視覺藝術」、「音樂與表演藝術」、「工藝」... 等 15 種產業類別：

上圖中，X (水平) 方向為年度 ( 由左而右，亦可設定為其它數值 )， Y (垂直) 方向則為用戶設定的數值。每一條線各代表一種類別 ( 如產業別中的「產品設計」)，類別類型則對應到多條折線。

資料快選與控制面板

在此視覺化中，我們將主要的設定項目集中在介面的上方，方便用戶快速尋找特定的功能，如下圖所示。例如，資料三個維度中，「年度」在趨勢線圖中已完整呈現，而另外兩個維度「分類法」與「數值種類」則透過下拉式選單提供給用戶自行決定。下圖中，用戶可透過「分組方式」、「Y 軸數值」來選擇欲讀取的資料集。

用戶亦可設定「X 軸數值」來呈現「年份」以外的數值，此時圖表會依 X 、Y 所指定的數值變形成散布圖。關於散布圖的細節將於後方描述。

  • 分組方式 - 快選資料統計的分組方式 ( 如「產業別」、「資本額」、「縣市」等 )
  • Y 軸數值 - 快選主要呈現的數值 ( 如「家數」、「營業額」等 )
  • X 軸數值 - 預設為「年份」，但亦可設定為其它數值。使用年份以外的數值時，圖表會變形成散布圖。
  • 檢視 - 切換不同檢視，包含預設的「數值」(折線圖)、「序位」(連接散布圖) 與「比例」(堆積長條圖)。
  • 下載 - 下載目前檢視之資料或圖檔 ( 見「資料引用」一節 )

趨勢的輔助檢視

由於趨勢線圖中折線的數量由分組類型而定，呈現不同的數據時折線圖有時會因為線條數量過多而不利閱讀。因此我們提供折線的三種變形圖表，供使用者以不同面向來觀察資料的趨勢。欲依需求切換圖表，使用者可透過點擊控制面板 ( 上圖 ) 中的「檢視」選項來切換不同圖表的檢視模式，其中包括「數值」、「序位」與「比例」檢視，或者直接透過選取 X 軸數值來啟動連接散布圖。以下介紹補助檢視的圖表類型。

Bump Chart / 凹凸圖

凹凸圖簡化圖表的「數值」部份，以排名替代數值，相當適合用來比較各組數據之間的高下。點擊控制面板中的「序位」來啟用凹凸圖模式。欲回到表現「趨勢」的「折線圖」模式，點擊「數值」檢視按鈕即可。

Stacked Bar Chart / 100% 堆積長條圖

100% 堆積長條圖以堆積色塊表現各組數值佔當年度數值總和的比例，適合用來觀察各組對於總體的貢獻程度。點擊控制面板中的「比例」來啟用堆積長條圖模式。欲回到表現「趨勢」的「折線圖」模式，點擊「數值」檢視按鈕即可。

Connected Scatter Plot / 連接散布圖

連接散布圖將 X 軸保留給額外一組數據使用，時序資訊則仍保留在視覺化的連線之中，適合用來觀察多組數據之間的相關性。

連接散布圖其實只是將散布圖中的點依分組連線，但由於比較少見，用戶可能較難理解，因此我們額外設計一個簡單的使用說明，在第一次切換至連接散布圖時彈出 ( 如右圖 )。

欲切換至連接散布圖模式，只需直接設定 X 軸為「年度」之外的數值即可。用戶可透過滑鼠選取並強調特定分類下的線條組，被選取的線條除顯示各點對應的年度外，並使用線條虛線的移動動畫提示用戶各點之間時間的先後關係。如下圖中，「出版」業為中央綠色的線條，周圍已有各點對應的年度標籤：

資料篩選功能

除了提供變形圖表供用戶以不同面向觀察資料之外，我們也加入資料篩選功能協助用戶適時簡化資料，將注意力聚焦在重點數據之上。

用戶可以點擊圖例來開關各個子分組，藉此簡化呈現的分組數量。我們亦提供「反向選取」、「全選」等輔助式的機制，減少操作篩選功能所需的時間。用戶亦可將滑鼠移至資料線段上強調特定資料分組。

下圖中對應的欄位

  • A - 可點擊的圖例，用來切換特定分組是否於圖表中顯示
  • B - 快選功能，可以重置選項或是反向選取
  • C - 使用滑鼠可強調特定資料分組。其它未被強調的分組會淡化處理。

比較特定年度

當用戶需要從趨勢進一步比較特定年度中各分組的數據時，可以點擊圖表中該年度之區間，進入「特定年度檢視」模式。在這個檢視模式下，我們移除了其它年度的資訊，而進一步我們可以使用更多樣的形式呈現資料。其中包含了以下幾種檢視：

長條圖

將各組數值以長條表現，並依數值大小預作排序。適合做各組數值間的比較。

熱度地圖

僅適用於「縣市」分組，協助用戶觀察資料的地理分布。

泡泡圖

亦用於比較各組數據大小，但較為活潑，版面配置亦較為方正。

控制面板

如同逐年趨勢檢視，在特定年度檢視模式下，我們亦將各個功能與控制項目集中在面板上方，方便用戶快速切換設定。其中包含了以下項目：

  • 回到線圖 - 退出特定年度檢視模式，回到趨勢線圖檢視
  • 動畫 - 以動畫播放年度資訊 ( 見「動畫檢視」一節 )
  • 各圖表檢視切換 - 包括長條圖、地圖、泡泡圖等
  • 顯示數值 - 供用戶快速切換欲呈現的數值 ( 如營業額、家數等 )
  • 下載 - 下載目前檢視之資料或圖檔 ( 見「資料引用」一節 )

動畫檢視

特定年度檢視下，單純使用靜態圖表並無法呈現時序的資料，但此時時序資料仍可以透過上述圖表搭配動畫來表現。由於動畫長條圖與動畫泡泡圖戲劇性的變化效果，相較於表現「趨勢」為重點的折線圖來說，動畫長條 / 泡泡圖更能向用戶強調「數值變化」的概念。

欲啟用動畫效果，用戶可點擊「控制面板」之動畫控制鈕，或透過圖表下方之時間軸控制項自由拖曳當前時間點來瀏覽。


資料引用

為協助用戶於網站以外的地方活用本視覺化，我們提供「資料下載」功能。 用戶可於介面中依需求選取包含「PNG」、「SVG」、「CSV」以及「全部」的四種選項，並運用於各種情境。

  • PNG - 點陣圖檔，可用於簡報、網站等情境。
  • SVG - 向量圖檔，可依需求使用向量圖編輯軟體做調整。
  • CSV - 檢視中的數據資料檔。可匯入試算表軟體中做資料分析。
  • 全部 - 包含上述三者的壓縮檔。
